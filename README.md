# 🚀 Go Web App – DevOps Practice Project

This project is a hands-on DevOps practice implementation using a simple Go-based web application.  
---

## 📌 Project Goal

The main goal of this project is to understand and practice real-world DevOps workflows such as:

- Building and running a Go web application
- Containerizing the application using Docker
- Understanding and implementing CI/CD pipeline structure
- Deploying workloads using Kubernetes manifests

---

## 🧰 Tech Stack

- Go (Golang)
- Docker
- Kubernetes (YAML manifests)
- Git & GitHub
- CI/CD

# ⚙️ How to Run Locally

## 🚀 Go Web App – Local Setup Guide

---

## 1️⃣ Clone the repository

```bash
git clone https://github.com/Reshmavinu/go-web-app-devops-practice.git
cd go-web-app-devops-practice
```

---

## 2️⃣ Run the application

### 🔨 Build the project

```bash
go build -o main .
```

### 📌 What it does:
- Compiles the Go code in the current folder
- Creates an executable file named `main`

---

### ▶️ Run the application

```bash
./main
```

---

## 🌐 Application URL

Open in browser:

**http://localhost:8080**
